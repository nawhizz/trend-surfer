  2. 주가정보 (daily_candles) — 경로에 따라 다름

  (A) 당일 시세 수집 — KRX 공식 API ⭐ daily_routine이 쓰는 메인 경로
  - collector.fetch_daily_ohlcv() → krx_collector.fetch_market_ohlcv_by_date() (krx_collector.py:24)
  - requests.post로 KRX 공식 API 직접 호출 (/stk_bydd_trd=KOSPI, /ksq_bydd_trd=KOSDAQ)
  - 과거엔 FDR을 썼으나 당일 404 빈발로 KRX 공식 API로 전환했습니다.

  (B) 과거 데이터 백필 — 하이브리드 (FDR + KRX)
  - hybrid_collector.backfill_hybrid() (hybrid_collector.py:2,71)
  - FDR: OHLCV(수정주가) → fdr.DataReader(ticker, ...)
  - KRX 공식 API: 거래대금(amount), 시가총액(market_cap) → FDR에 없는 값 보강
  - FDR 실패 시 KRX 데이터로 폴백(L104~).
  - update_adjusted_prices.py의 수정주가 백필도 이 경로를 사용합니다.
